Tool Evaluation Questions with Wicket vs Angular Case Study

  • What kind of community support exists?

    Stackoverflow post counts

    • Wicket: 2389
    • AngularJS: 51,877

    Mailing lists

    Wicket: http://wicket-users.markmail.org/

  • How easy is it to learn?

    Wicket Quick Start: Grow on top of Java with maven applying Swing knowledge.
    Angular Quick Start: Grow on top of Javascript with node.

    Google search for tutorials:

    • AngularJS tutorial: About 1,180,000 results
    • Wicket tutorial: About 242,000 results

  • How easy is it to understand?

    Multiple Layers

    Would you like to learn Wicket, the translation to javascript from Wicket, and javascript?

    Similar issue with Hibernate

  • Does it support the desired level of program complexity?

    Basic or Complex?

    Maybe your current program is very simple, but...

  • Does it supply the right level of “magic”?

    Attempt to hide all the javascript? Too much magic.

    Let me define a variable in javascript and have it available in the HTML (two-way data binding)? Good magic.

  • How fun is it to use?

    I have NEVER heard anyone describe Wicket as "fun"
